# 🐍 Neon Snake Engine

🚀 [Play the Game](https://neon-snake-engine.vercel.app)

A modern and interactive Snake Game built using **HTML, CSS, and JavaScript**, enhanced with smooth animations, dynamic difficulty, and a polished UI.

This project goes beyond a basic implementation by focusing on **game logic, performance, and user experience**.

---

## ✨ Features

- 🎮 Classic Snake gameplay with smooth controls
- ⚡ Dynamic difficulty (speed increases as score grows)
- 🧠 Collision detection (wall + self)
- 🍎 Smart food generation (avoids spawning on snake)
- 🏆 High score tracking using LocalStorage
- ⏱ Real-time timer system
- 🎨 Modern UI (glassmorphism + neon effects)
- 🔄 Smooth restart system without bugs

---

## 🛠 Tech Stack

- HTML5
- CSS3 (Grid, Animations, Glassmorphism)
- Vanilla JavaScript (ES6)
