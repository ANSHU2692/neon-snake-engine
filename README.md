# 🐍 Neon Snake Engine

A modern, interactive Snake Game built using **HTML, CSS, and JavaScript**, enhanced with smooth animations, dynamic difficulty, and a polished UI.

This project goes beyond a basic Snake Game — it focuses on **game logic optimization, UI/UX improvements, and scalable structure**.

---

## 🚀 Features

- 🎮 Classic Snake gameplay with smooth controls
- ⚡ Dynamic difficulty (speed increases as score grows)
- 🧠 Collision detection (wall + self)
- 🍎 Smart food generation (avoids spawning on snake)
- 🏆 High score tracking using LocalStorage
- ⏱ Real-time timer system
- 🎨 Modern UI (glassmorphism + neon effects)
- 🧩 Modular and clean code structure
- 🔄 Restart system without memory leaks

---

## 🛠 Tech Stack

- HTML5
- CSS3 (Animations, Grid, Glassmorphism)
- Vanilla JavaScript (ES6)
